What is HTML?

HTML stands for HyperTextMarkupLanguage. It's a markup language that web developers use to structure and describe the content of a webpage (not a programing language)

HTML consists of elements that describe different types of content: paragraphs, links heading, images, video, etc. Web browsers understeand HTML and render HTML code as websites. p>

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. the closing tag
  3. The actual element
